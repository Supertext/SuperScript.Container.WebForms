What Does It Do?

This library is an add-on for SuperScript. It is strongly recommended to read the documentation for the core project before proceeding with this library. If you haven't yet read the core documentation then it is recommended that you start by reading about the core concepts.

SuperScript.Container.WebForms contains the core code used by higher-level, more functional SuperScript projects which offer a WebForms .aspx container control for enclosing server-side entities to be processed by SuperScript. 

	public abstract class Container : System.Web.UI.WebControls.PlaceHolder
	{
	    public bool AddLocationComments { get; set; }
	    public string EmitterKey { get; set; }
	    public int? InsertAt { get; set; }
	    protected abstract void __PreRender(object sender, EventArgs e)
	    protected abstract string GenerateComment(string comment, bool startOnNewLine = true)
	    protected string GetFileName()
	    protected virtual string GetContents()
	}

The above abstract class contains the base functionality for retrieving the contents of a derived container control, allowing the derived implementation to move the contents into an implementation of DeclarationBase.

Name Type Description
AddLocationComments bool

Determines whether the emitted contents contain comments indicating the original location when in debug mode.

Optional. The default value is true.

EmitterKey string

Indicates which instance of IEmitter the content should be added to.

Optional. If not specified then the contents will be added to the default implementation of IEmitter.

Read more about the default emitter here.

InsertAt Nullable

Gets or sets the index into which the resultant instance of CollectedScript will be inserted into the central Collection of declarations.

Optional. Default value is null which appends the resultant instance of CollectedScript to the central Collection of declarations.

__PreRender void

This abstract method presents a location in derived types where the contents of the container control may be retrieved.

This is also the lcoation where the contents of the container control are removed or replaced.

GenerateComment string

This abstract method presents a location in derived types to generate a comment block, formatted for the specific output type of the derived implementation.

GetFileName string

Returns the relative path and filename of the control or page which is instantiating this control.

Tyically this will be called by the derived implementation of GenerateComment.

GetContents

Obtains the contents that have been passed into this container control.

Typically this will be called by the derived implementation of __PreRender.
